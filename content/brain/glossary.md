# Glossary

**Purpose.** Acronyms and terminology used across Toggle work. Seeded from terms that appear in case studies — extend as new ones come up.

| Term | Meaning |
|---|---|
| TOF | Top of Funnel — awareness / curiosity-stage messaging |
| MOF | Middle of Funnel — consideration / reassurance-stage messaging |
| BOF | Bottom of Funnel — conversion / urgency-stage messaging |
| CPL | Cost Per Lead |
| CPA | Cost Per Acquisition |
| ROAS | Return On Ad Spend |
| GMV | Gross Merchandise Value — used on TikTok Shop, marketplaces |
| YMYL | Your Money or Your Life — Google content category covering finance, health, safety; held to higher quality bars |
| E.E.A.T. | Experience, Expertise, Authoritativeness, Trustworthiness — Google's quality framework, decisive in YMYL verticals |
| CAPI | Conversions API — server-side conversion signal pipe (Meta) for stronger attribution post-iOS14 |
| PMax | Performance Max — Google's automated multi-channel campaign type |
