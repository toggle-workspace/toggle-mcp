# /landing-page

Draft a landing page copy + structure for one client.

## READS
- brain/positioning/                       # positioning + differentiators
- brain/voice/                             # house voice
- clients/<slug>/style-pack.md             # client voice overrides

## WRITES
- clients/<slug>/01-strategy/landing-page-YYYY-MM-DD.md

## INPUTS
- $slug — the client slug
- $offer — what the page is selling
- $audience — primary audience for the page

## STEPS
1. Read positioning + brain voice rules.
2. Read the client's style-pack for overrides.
3. Draft section-by-section: hero, problem, solution, proof, objections, CTA.
4. Note above-the-fold priorities and conversion intent per section.
5. Write to `clients/$slug/01-strategy/landing-page-YYYY-MM-DD.md`.
